# LineDog Rest

Mac 菜单栏护眼桌宠：**整点/半点或手动番茄** 触发 5 分钟休息霸屏；**自然语言**通过 Gemini 写入系统**提醒事项**；另有**可配置时长**的独立倒计时铃铛。需求与范围见 [PRD.md](PRD.md)。

## 用 Xcode 打开与运行

1. 双击打开 **`LineDog.xcodeproj`**。
2. 选择 Scheme **LineDog**，运行目标为 **My Mac**（⌘R）。
3. 应用为 **仅菜单栏**（`LSUIElement`），Dock 中无图标；点菜单栏 **小狗图标** 打开控制面板（与 **左键点桌宠** 相同界面）。

### 全局快捷键

均在 **LineDog 设置** 中可改键位；默认使用 **Carbon 全局热键**，一般 **无需**「隐私与安全性 → 辅助功能」。

| 默认 | 作用 |
|------|------|
| **⌘⇧<** | 唤起 **智能输入**（自然语言添加提醒） |
| **⌘⇧.** | 弹出 **控制面板**（同菜单栏） |
| **⌘⇧M** | **独立倒计时提醒**：未在倒计时则开始，进行中则取消 |
| **⌥⌘R**（可选） | 备用唤起智能输入；需对 LineDog 开启 **辅助功能**（`NSEvent` 全局监听） |

### 智能输入（Gemini → 提醒事项）

- 在 **设置** 中填写 **Gemini API Key**，并选择 **模型**（如 2.5 Flash、2.5 Pro、3.x 预览等；列表随 Google 开放情况维护）。
- 模型返回 **单个 JSON 对象** 或 **对象数组** 时，客户端会 **创建一条或多条** 提醒。
- 用户只说日期、没说钟点时：由模型 + 客户端规则 **推断** `hour`/`minute`（避免一律深夜或占位无效时间）。
- 输入框 **关掉面板后字会保留**；**只有提醒真正保存成功**才会清空草稿（失败可继续改后再试）。
- 自然语言含 **每天/每周/每月…** 时，JSON 中的 `recurrence` 会写入系统提醒的 **重复**（EventKit）；`alarm_date` 表示 **下一次** 截止时间。

### 独立倒计时提醒

- 在 **菜单栏/桌宠控制面板** 里用 **Stepper** 设置 **1～180 分钟**（默认 7），再点「开始 *N* 分钟倒计时」或使用 **⌘⇧M**。
- **全局快捷键** 的改键仍在 **设置** →「独立倒计时提醒（快捷键）」。

### 提醒事项权限：为什么每次编译都要再点一次「允许」？

系统 **不允许** App 自动勾选或绕过提醒权限，必须由你在系统对话框里确认（和日历、麦克风一样，属于 TCC）。

若你发现 **每 ⌘R / 重新编译一次就要再授权**，多半不是代码问题，而是 **调试签名不稳定**：Xcode 未指定 **Team** 时，常用临时/ ad hoc 签名，**每次构建主程序哈希会变**，macOS 会像对待「新应用」一样再次弹窗。

**建议**：Target **LineDog** → **Signing & Capabilities** → 勾选 **Automatically manage signing**，**Team** 选你的 **Apple ID（Personal Team）** 或开发者账号。保持 **Bundle Identifier** 不变（工程里为 `com.linedog.LineDog`）。这样授权会记在 **系统设置 → 隐私与安全性 → 提醒事项** 里，一般不必每编一次点一次。

从 macOS 14 起，若系统已记录为已授权，应用侧会先查 `authorizationStatus` 再决定是否调用 `requestAccess`，避免多余弹窗。

### 桌宠与多显示器

- 全程 **只有一只小狗**：默认在 **菜单栏所在物理显示器** 右下角（`CGMainDisplayID` 匹配 `NSScreen`）。休息时 **同一只** 变红并移到中央，背景渐暗。
- 插拔或调换主屏后会短暂防抖再对齐；窗口不使用 `fullScreenAuxiliary`，以免双屏合成异常。

## 使用说明

启动后**默认**为 **整点 / 半点** 模式；可在面板里切到 **手动番茄**。

| 模式 | 行为 |
|------|------|
| **手动番茄** | 点「开始专注」后计时 25 分钟，结束后 5 分钟休息霸屏，再进入下一轮 25 分钟。 |
| **整点 / 半点** | 在每小时 `:00` 与 `:30` 触发 5 分钟休息霸屏。 |

计时进行中显示 **「停止计时」**；暂停后为 **「恢复计时」**。休息期为 **无边框全屏置顶**，无关闭按钮；结束进程请用面板中的 **「退出应用」**。

- **「立即开始休息（测试）」**：立刻走 5 分钟霸屏，**不改动**当前番茄/自动锚点。
- **「休息期间阻止点击桌面」**：控制休息时是否挡住背后窗口的鼠标。
- **「出现 5 分钟小猫」**：独立陪伴小窗，与桌宠分层。

## 自定义线条小狗素材

在 Xcode 中打开 **`LineDog/Assets.xcassets`**，向 **`LineDogPet`** 图像集拖入 PNG/PDF（建议透明底）。未配置时使用系统符号 `dog.fill` 作为占位。

## 工程结构（对应 PRD 模块化）

- **`TimerEngine/`**：`TimerEngine` 协议 + `ManualTimerEngine` / `AutoTimerEngine`，统一产出 `TimeState`。
- **`WindowManager/`**：桌宠 `NSWindow` + `PetStageView`（常态与休息同一套视图与 `PetRenderer`）；智能输入与 Toast 面板。
- **`PetRenderer/`**：桌宠图像布局与配色态。
- **`SmartReminder/`**：Gemini 客户端、Prompt、JSON 解析（含多对象数组）、编排与 EventKit 写入、推断时刻、调试日志等。
- **`SevenMinuteReminder/`**、**`FiveMinuteCat/`**：独立倒计时与小猫陪伴窗口。

## 命令行编译（可选）

```bash
cd /path/to/LineDog
xcodebuild -scheme LineDog -configuration Debug -destination 'platform=macOS' build
```

若需在沙箱或 CI 中指定产物目录，可增加 `-derivedDataPath ./DerivedData`。
