
### PRD: Mac 桌面端「线条小狗」护眼桌宠 (LineDog Rest)

#### 1. 产品概述
* **产品名称**: LineDog Rest
* **产品定位**: 一款弱交互、强提醒的 Mac 状态栏/桌面后台应用：番茄休息节奏 + 可选的自然语言提醒写入。
* **核心目标**: 通过不可忽略的全屏动画，推动「工作 / 休息」护眼循环；并支持用口语快速创建系统「提醒事项」。

#### 2. 技术栈约束
* **开发语言**: Swift 5+
* **UI 框架**: SwiftUI（状态栏菜单、设置）+ AppKit（透明置顶桌宠窗、智能输入/气泡等 `NSPanel`）。
* **架构**: MVVM / 单向数据流；计时、窗口、渲染分层（见第 4 节模块划分）。

#### 3. 核心功能需求

**3.1 运行模式（双模式引擎）**
* **模式 A：手动番茄钟** — 从「开始专注」起 25 分钟工作 → 5 分钟霸屏休息 → 再进入下一轮 25 分钟。
* **模式 B：系统时钟锚定** — 每逢 `:00` 与 `:30` 触发 5 分钟霸屏休息（例如 10:25 进入则 10:30 触发）。

**3.2 小狗霸屏（休息期视觉）**
1. 右下角出现桌宠 → 2. 放大并背景渐暗 → 3. 视觉中心保持至休息结束 → 4. 渐隐恢复。
* 窗口层级：`NSWindow.Level.screenSaver` 等，覆盖 Dock 与菜单栏区域。
* 无传统关闭按钮；可通过状态栏菜单「退出应用」结束进程。
* **可选项**：「休息期间阻止点击桌面」开关（开则霸屏挡鼠标，关则透传背后操作）。

**3.3 桌宠与入口**
* **仅菜单栏**（`LSUIElement`）；**一只**桌宠，默认在**菜单栏所在屏**的右下角，可拖动，位置持久化。
* **左键**桌宠 / **⌘⇧.**（默认，可改）：弹出与菜单栏相同的控制面板。
* **右键**桌宠：唤起**智能输入**（添加提醒）。
* 多显示器：桌宠几何对齐主菜单栏屏，插拔屏防抖重定位。

**3.4 智能输入（自然语言 → 提醒事项）**
* 使用 **Google Gemini** `generateContent`，结构化 JSON → 解析后写入 **EventKit 提醒事项**。
* **设置**：Gemini API Key（本机 UserDefaults）、**可选模型**（如 2.5 Flash / Pro、3.x 预览等，以官方开放为准）。
* **行为要点**：
  * 支持 **单条 JSON 对象** 或 **对象数组**：一句话多条独立待办时，**批量创建**多条提醒。
  * `alarm_date`：用户未说钟点时，**模型推断** + 客户端 **`SmartReminderInferredTime`** 兜底（`hour`/`minute` 均为 null 时）。
  * 支持 **重复**（`recurrence` → `EKRecurrenceRule`）、**日常**标签与备注策略（`#日常` 等）。
  * 解析成功但 **无法写入 EventKit** 时，**不**用整句原文冒充「普通备忘」成功；须明确失败提示。
  * 输入框 **草稿**：点外部 / Esc /「取消」关闭面板后**保留**；**仅在实际写入成功**且草稿未被用户改掉时清空。
* **全局快捷键 ⌘⇧<**（默认，可改）：唤起智能输入；再按可关闭（与实现一致）。
* **⌥⌘R**（可选）：依赖「辅助功能」授权的全局按键，同样唤起智能输入。

**3.5 独立倒计时提醒（非番茄休息）**
* 与桌宠分层：右下角小倒计时条，结束后面板中央铃铛 + 文案，点击关闭。
* **时长**：在**菜单栏/桌宠控制面板**内用 Stepper 设置 **1～180 分钟**（默认 7），写入 UserDefaults。
* **全局快捷键**（默认 **⌘⇧M**，可在设置中修改）：进行中则**取消**，否则**开始**；Carbon 注册，一般无需辅助功能。

**3.6 其他独立功能**
* **5 分钟小猫陪伴**：独立窗口，与桌宠分层展示，时间到渐隐。
* **立即开始休息（测试）**：仅走霸屏 UI，不破坏当前番茄/自动锚点状态。

**3.7 性能与刷新**
* 计时引擎在**锚点等待**阶段避免重复派发相同状态；专注/休息倒计时按**整秒变化**更新 UI，降低无效 `@Published` 与 WindowServer 负载。
* 休息霸屏阶段桌宠位移动画使用 **约 15Hz** 定时刷新（原为 60Hz），在观感与 CPU 间折中。

#### 4. 模块化架构（实现对应）

* **模块 1: `TimerEngine`** — `TimerEngine` 协议；`ManualTimerEngine` / `AutoTimerEngine` 统一产出 `TimeState`。
* **模块 2: `WindowManager`** — 桌宠 `NSWindow` 生命周期、休息霸屏、智能输入/Toast `NSPanel`、与 `PetStageView` 协同。
* **模块 3: `PetRenderer`** — 桌宠静态图/SF Symbol 布局与配色态（运行黑 / 休息红 / 暂停白描边 / 智能输入思考态等）。
* **模块 4: `SmartReminder/`** — Prompt、Gemini 客户端、JSON 解析（含数组）、Orchestrator、EventKit 写入、推断时刻、调试日志等。
* **模块 5: 全局快捷键** — `LineDogCarbonGlobalHotKeys`（Carbon `RegisterEventHotKey`）+ 设置内持久化键位。
